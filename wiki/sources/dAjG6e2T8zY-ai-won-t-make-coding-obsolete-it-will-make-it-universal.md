# AI won’t make coding obsolete. It will make it universal.

## Metadata

- Video ID: `dAjG6e2T8zY`
- Channel: Y Combinator
- Published: 2025-08-28T17:36:02Z
- URL: https://www.youtube.com/watch?v=dAjG6e2T8zY

## Summary

The speaker argues against the notion that AI will render coding obsolete, instead positing that AI tools will democratize software engineering, making it a universal skill that enhances productivity across all professional roles.

## Key Ideas

- Historical parallels exist between the introduction of high-level programming languages and current AI fears.
- Coding is a force multiplier for non-technical roles like finance and recruiting.
- Lowering the barrier to entry for programming increases the total number of people who should learn to code, rather than decreasing it.
- The future of work involves empowering everyone to leverage coding skills to improve their specific job functions.

## Notable Claims

- Advising people not to learn to code because of AI is poor career advice. Evidence: Historical precedents show that easier coding tools (like high-level languages) have historically increased, not decreased, the demand for programmers.
- Coding skills improve performance in non-technical roles. Evidence: The speaker observes that their own CFO, head of talent, and recruiters perform better in their specific functions because they possess coding skills.

## Quotes

> As better tools make software engineering easier, more people should do it, not fewer.
> I think it's time for everyone of every job role to learn to code.
> In the future, I think we empower everyone to code.
