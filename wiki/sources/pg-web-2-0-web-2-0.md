# Web 2.0

## Summary

Paul Graham analyzes the term 'Web 2.0' in 2005, arguing that while the term was initially meaningless marketing jargon, it evolved to represent a shift toward web-based applications (Ajax), user-driven democracy, and a more user-centric design philosophy. He concludes that 'Web 2.0' essentially describes using the web as it was naturally intended to be used, with Google serving as the primary example of this alignment.

## Key Ideas

- Web 2.0 is a retrospective label for the web finding its 'natural angle of repose' after the dot-com bubble.
- Ajax enables web applications to function like desktop software, marking a significant shift in development.
- Web democracy (Wikipedia, Reddit, Digg) demonstrates that amateur-driven systems can outperform traditional editorial models.
- Successful modern startups prioritize user experience and generosity over the high-handed, restrictive practices of the 90s.
- The 'Web 2.0' trend is not a new invention but a return to the web's inherent potential, which was previously obscured by broken business models.

## Topics

- [[topics/ajax|Ajax]]: A set of web development techniques that allow web pages to update asynchronously, enabling desktop-like application performance.
- [[topics/web-democracy|Web Democracy]]: The shift toward user-generated content and community-driven curation as a superior alternative to traditional editorial gatekeeping.
- [[topics/user-centric-design|User-Centric Design]]: The philosophy of treating users with generosity, avoiding unnecessary registration, and prioritizing utility over aggressive monetization.
- [[topics/dot-com-bubble|Dot-com Bubble]]: The period of market speculation in the late 90s that imposed broken business models on the web.

## Notable Claims

- The 'Web 2.0' label is essentially meaningless marketing that only gained substance because it happened to coincide with the rise of Ajax and user-driven platforms. Evidence: Graham notes that he despised the term but realized it had become a shorthand for 'doing things right' on the web.
- Selection beats damping in content production. Evidence: The vast pool of online writers, combined with community-driven selection mechanisms, produces higher quality content than editor-damped print media.
- The default exit strategy for modern startups is acquisition rather than IPO. Evidence: The IPO market is largely closed, and acquirers are more rational than retail investors.

## Quotes

> Web 2.0 means using the web the way it's meant to be used.
> Selection beats damping, for the same reason market economies beat centrally planned ones.
> If you find yourself asking 'should we allow users to do x?' just answer 'yes' whenever you're unsure. Err on the side of generosity.
