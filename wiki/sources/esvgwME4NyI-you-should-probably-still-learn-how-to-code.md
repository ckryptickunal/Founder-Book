# You should probably still learn how to code.

## Metadata

## Summary

The speaker argues that despite the rise of AI-driven natural language programming, learning to code remains essential because the process of programming fundamentally enhances human logical thinking and cognitive development.

## Key Ideas

- Programming is a cognitive exercise that improves logical reasoning.
- AI models (LLMs) demonstrate that learning to code is a foundational step for achieving logical intelligence.
- The goal of technology should be to make programming unnecessary, but the human benefit of learning it persists.
- There is empirical evidence linking code comprehension to the development of logical thinking in AI, which mirrors the human experience.
