## Summary

Jay Rosenkrantz, co-founder of Figments, explains how the company is building the 'WWE for esports' by creating virtual characters with fictional storylines that compete in popular video games. The company leverages real-time game engines, motion capture, and remote collaboration tools to produce live-streamed entertainment, successfully partnering with brands like Reebok and Guayaki.

## Key Ideas

- Creating virtual influencers that compete in esports to build entertainment franchises.
- Developing proprietary virtual production tools that allow for remote motion capture and live animation.
- Utilizing real-time game engines like Unreal Engine and ARKit for scalable, home-based content creation.
- Transitioning from a studio-based model to a remote-first model to access global talent.
- Integrating brand sponsorships directly into virtual character storylines and live streams.
