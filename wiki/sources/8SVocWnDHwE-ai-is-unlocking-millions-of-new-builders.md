# AI Is Unlocking Millions Of New Builders

## Summary

Mukund and Madhav Jha, founders of Emergent, discuss their platform's rapid growth in enabling non-technical users to build and deploy production-ready software using AI agents. The conversation covers their pivot from enterprise-focused coding agents to a consumer-facing tool, their unique infrastructure approach, and the broader societal impact of democratizing software creation.

## Key Ideas

- Software development is shifting from a technical barrier to a creative expression for domain experts.
- Emergent uses a multi-agent architecture with long-term memory and autonomous verification to handle end-to-end production deployment.
- The platform prioritizes 'agent experience' and user empathy, abstracting complex technical concepts like API keys and Git for non-technical users.
- There is a growing trend of 'solopreneurship' where individuals build niche, high-value applications that were previously economically unviable.
- The 'Jevons paradox' applies to software: as tools become more powerful, the demand for software and the complexity of projects increase rather than decrease.
- Building a global company from India with a US-centric user base requires deep customer empathy and a 'global-first' mindset.

- 80% of Emergent users are non-technical with zero programming knowledge. Evidence: Stated by the founders during the interview regarding their current user base.
- 7 million apps have been built on Emergent in 8 months. Evidence: Reported growth statistics since the platform's launch.
- The cost to build custom software has dropped from $500,000 to $5,000 using AI platforms. Evidence: Founders' estimate of the economic value provided to small business owners.
- Software engineering jobs are not disappearing, but the market is expanding. Evidence: Observation of increased demand for software and higher productivity per individual.
