# Value is created by doing

## Metadata

- Video ID: `sa-value-is-created-by-doing`

## Summary

Sam Altman argues that true value creation stems from tangible execution rather than performative activities like networking, reading industry news, or discussing ideas. He emphasizes that while strategy is essential, it is worthless without implementation, and advises individuals to focus on high-impact work that aligns with their genuine passions.

## Key Ideas

- Value is created through execution, not through networking, meetings, or industry chatter.
- Strategy is important, but execution is where value is captured.
- Productivity should be measured by tangible outcomes like revenue or closed deals, not by the number of meetings or hours spent.
- It is better to join a great company working on a mission you care about than to start a mediocre company.
- Individuals should prioritize working on what they are truly passionate about rather than delaying their goals for financial security.

## Notable Claims

- Most activities that feel like work, such as tweeting or networking, do not create new wealth. Evidence: Altman notes that value is created by building and selling products to customers.
- Measuring productivity by the number of meetings leads to more meetings, not more output. Evidence: Altman suggests measuring productivity by revenue growth or closed investments instead.
