## Summary

Jimmy, co-founder and CTO of Pocket Worlds, explains the 'social-first' design philosophy behind their MMO app Highrise. The platform prioritizes social features like messaging and community building over traditional game mechanics to ensure long-term user retention, utilizing a hybrid architecture that combines native mobile UI with a C++ game engine.

## Key Ideas

- Social-first design prioritizes community, messaging, and profiles over traditional game loops.
- Long-term retention in mobile games is driven by social connections rather than gameplay mechanics.
- Native UI (Swift/Kotlin) is superior to game engine UI for creating social-app experiences.
- A hybrid architecture (50% native UI, 50% C++ game engine) allows for modular game development.
- The 'social-first' framework can be reused to build diverse game genres (e.g., hero collectors, adventure games) using the same underlying social infrastructure.

## Topics

- [[topics/social-first-design|Social-First Design]]: A development philosophy that builds the game around social infrastructure (messenger, guilds, profiles) rather than adding social features as an afterthought.
- [[topics/hybrid-architecture|Hybrid Architecture]]: A technical approach combining native mobile UI (Swift/Kotlin) for user experience and C++ for game engine logic to achieve both app-like feel and game-like performance.

## Notable Claims

- Native UI (Swift/Kotlin) provides a better user experience for social apps than game engines like Unity. Evidence: Apple and Google have spent decades perfecting fonts and UI animations that game engines cannot replicate.
- Social-first design solves the problem of long-term player retention in mobile games. Evidence: Players join for the game but stay for the social connections and community.

## Quotes

> With the social first philosophy, what we do is we start with the messenger, a news feed for a community, guilds, profiles, an economy and a vibrant world, and then we add in the game.
> A player joins Highrise for the design game, but they really ultimately stay for the friends they meet in our world.
