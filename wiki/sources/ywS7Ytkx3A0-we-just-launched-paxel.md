## Summary

Y Combinator introduces Paxel, a local analysis tool that evaluates how developers build software using AI agents by analyzing their coding sessions and providing a builder profile.

## Key Ideas

- Software development with AI is currently an isolated, poorly understood process.
- Paxel provides a 'builder profile' based on five dimensions: steering, execution, engineering, product instinct, and planning.
- The tool operates locally within Docker to ensure code privacy and security.
- Paxel profiles can be used as a supplement for Y Combinator Startup School applications to demonstrate actual building habits.
- The goal is to identify talented builders who might not be captured by traditional resume-based screening.
