# Internal Agent Builder

## Metadata

## Summary

Y Combinator is seeking founders to build 'internal agent builder' infrastructure, tools that allow employees to automate repetitive tasks by integrating with their existing software stacks and securely interfacing with LLMs.

## Key Ideas

- The future of work involves every employee creating custom agents to handle mundane tasks.
- Internal agent builders require deep integration with existing enterprise software.
- Security, permission management, and data privacy are critical components of agent infrastructure.
- Automating repetitive workflows allows employees to focus on high-value, human-centric work.

## Entities

- [[entities/y-combinator|Y Combinator]] (organization): A startup accelerator that provides seed funding for startups.

## Topics

- [[topics/ai-agents|AI Agents]]: Software entities designed to perform repetitive tasks autonomously on behalf of employees.
- [[topics/enterprise-automation|Enterprise Automation]]: The integration of AI tools into existing business workflows to increase productivity.

## Notable Claims

- Every employee will soon build their own agents to automate repetitive parts of their jobs. Evidence: Y Combinator's observation of internal efficiency gains from using their own agent-building tools.
- Internal agent builders can effectively handle tasks like reviewing term sheets and accounting workflows. Evidence: Y Combinator's internal implementation of these tools for their own operations.
