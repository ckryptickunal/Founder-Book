# RFS: Better Enterprise glue

## Summary

The video identifies the 'dark matter' of enterprise software—the massive amount of custom, invisible code required to integrate purchased software with existing internal systems—and proposes that LLMs can automate the creation and maintenance of this integration glue.

## Key Ideas

- Enterprise software requires significant custom code to connect with internal systems like data warehouses and HR databases.
- This 'dark matter' of integration is company-specific, invisible, and requires constant maintenance.
- Large software vendors rely on multi-billion dollar ecosystems of consultants to build these custom integrations.
- LLMs offer a transformative opportunity to automate the generation and maintenance of this custom integration code.
